# Community Projects

The following projects are built and maintained by the community. We appreciate all contributions! Note that these projects are not officially supported by the BantuVoice team.

If you have a project you'd like to add, please open a PR.

---

- **[ComfyUI-BantuVoice-TTS](https://github.com/Saganaki22/ComfyUI-BantuVoice-TTS)** —
  ComfyUI custom node for BantuVoice text-to-speech generation.

- **[vLLM-Omni](https://github.com/vllm-project/vllm-omni)** —
  A framework for efficient model inference with omni-modality model. Supports BantuVoice serving.

- **[pyVideoTrans](https://github.com/jianchang512/pyvideotrans)** —
  Video translation tool with dubbing & subtitles. Supports BantuVoice as a TTS engine.

- **[MLX-Audio](https://github.com/Blaizzy/mlx-audio)** —
  TTS, STT, and STS library built on Apple's MLX framework. Supports
  BantuVoice among other models for efficient speech processing on Apple Silicon.

- **[RealtimeTTS](https://github.com/KoljaB/RealtimeTTS)** —
  Converts text to speech in realtime. Supports BantuVoice as a TTS engine.

- **[TTS-WebUI](https://github.com/rsxdalv/TTS-WebUI)** —
  Gradio web UI for multiple TTS models. Supports BantuVoice as one of its backends.

- **[BantuVoice-Studio](https://github.com/debpalash/BantuVoice-Studio)** —
  Desktop application for BantuVoice voice generation.

- **[bantuvoice-server](https://github.com/maemreyo/bantuvoice-server)** —
  OpenAI-compatible HTTP server for serving BantuVoice via `/v1/audio/speech`.
  Supports voice profiles for persistent cloning, sentence-level streaming,
  and optional Bearer auth.

- **[bantuvoice-rs](https://github.com/FerrisMind/bantuvoice-rs)** —
  GPU-first Rust workspace for BantuVoice inference, parity validation, CLI
  execution, and an OpenAI-compatible HTTP server built with Candle.

- **[bantuvoice-trtllm](https://github.com/tlitech/bantuvoice-trtllm)** —
  Deploy BantuVoice TTS model using TensorRT-LLM and Triton Inference Server
  on Modal, faster than PyTorch.
